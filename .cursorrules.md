# Project: AcadMatch AI (P2P Learning Platform)

## Tech Stack
- **Frontend:** React (Vite), Tailwind CSS, Lucide React (Icons), React Router DOM.
- **Backend/DB:** Firebase (Auth, Firestore, Hosting).
- **State Management:** React Context API or Zustand (keep it simple).
- **Language:** TypeScript.

## Core Vibe & Design System
- **Aesthetic:** "Clean Academic." Use whitespace, rounded corners, soft blues (primary) and slate grays (text). Card-based UI.
- **Tone:** Professional, encouraging, and focused on clarity.

## Coding Standards
- Use functional components with hooks.
- separate logic from UI (custom hooks preferred).
- Use Firebase Modular SDK (v9+).
- STRICTLY TYPE everything in TypeScript.
- formatting: Prettier default.

## Data Schema (Firestore)
- `users`: { uid, name, email, avatar, skillsOffered: [], skillsNeeded: [], availability: {} }
- `matches`: { userIds: [uid1, uid2], score: number, status: 'pending'|'active' }
- `chats`: { matchId, messages: [{ senderId, text, timestamp }] }
